# Football Fantasy

A fantasy football web app with real-time rooms/leagues, drafting, and a marketplace built with React + Firebase. App will similar to NFL Draft but with soccer in mind. 
This is a work in progress app with mock stats and players

## Live Demo
- https://fifa-fantasy-4a7e3.web.app/  

## Tech Stack
- React + Vite
- Firebase Auth
- Firestore (real-time)
- Firebase Hosting

## Key Features
- Magic-link / email login
- Create & join rooms
- Draft system with timer + auto-pick
- Marketplace open/close windows
- Real-time roster and room updates
- Email notifications

### Web App
